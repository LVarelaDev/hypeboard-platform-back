# Hypeboard Platform - Backend

Backend desarrollado en **NestJS** que procesa un mock de datos de YouTube y expone un endpoint optimizado para consumo frontend.

## Descripción

Este servicio actúa como una capa de transformación sobre un payload crudo, aplicando reglas de negocio para entregar una "Cartelera de Hype Tecnológico" con:

- Datos limpios
- Fecha en formato amigable
- Nivel de Hype calculado
- Identificación de la "Joya de la Corona"

---

## Tecnologías

- Node.js
- NestJS
- TypeScript
